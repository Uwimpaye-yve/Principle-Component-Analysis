# Principal Component Analysis - African Economic Outlook

## Overview

This project implements principal component analysis from using numpy

## Dataset

the dataset used is the African Economic Outlook from kaggle. it includes missing values, non-numeric colums and economic indicators for African countries.

## Implemented 

-Data standardization 
-Data preprocessing
-Eigen decomposition
-projection of data

## Requirements(install before running)

-Python3
-pandas
-numpy

## How to run

-Open the notebook in Google colab
-Upload dataset file
-Run the cell to see the output
